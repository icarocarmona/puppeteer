## Inicializando projeto

    npm init

## Instalando Puppeteer

    npm i puppeteer

## Pré-configuração
 Antes de rodar o projeto setar as variáveis de ambiente  `LEGENDASTV_USER`  e `LEGENDASTV_PWS`

```sh
    Exemplo Windows: set LEGENDASTV_USER=meu_usuario
```

## Rodando o projeto
    node index.js
